FAQ

Stream

  • How does the controller display work? He is using a splitter for the controller, so that one cable plugs into the N64 and one cable plugs into an Arduino (which is connected to the PC via USB). A program reads the input from the Arduino and displays it on the screen. Siglemic's device was made by Cylon13, but you can also make your own!
  • Where are the splits? He doesn't need splits himself to know how well he is doing and the chat tends to get very negative whenever there's a positive split.
  • Why is he dying suddenly all the time? He's probably using his GameShark device and a code that allows him to press D-pad down to instantly lose all health.
  • How does he get the in-game timer for non-race/non-slide levels? See the previous question. He's using a GameShark code.
